Tic-Tac-Toe (English)

Description:
    Console-based Tic-Tac-Toe game written in Python. Two players take turns entering a cell number. 
    The game updates the board and checks for a win or draw after each move.

Features:
    3x3 board;
    Two-player mode (X and O);
    Input validation;
    Win detection;
    Draw detection;
    Exit option;
    Technologies;
    Python 3;

Run
``` bash
git clone https://github.com/YOUR_USERNAME/tic-tac-toe.git
cd tic-tac-toe
python Tic-Tac-Toe.py
```

Controls:
    Enter a number from 1 to 9 to make a move: 1 | 2 | 3 4 | 5 | 6 7 | 8 | 9;
    Enter 0 to exit.

Notes:
    Project demonstrates basic game logic, input handling, and code structure.

Крестики-Нолики (Русский)

Описание:
    Консольная игра "Крестики-Нолики", написанная на Python. Два игрока по очереди вводят номер клетки. 
    После каждого хода поле обновляется и проверяется победа или ничья.

Возможности:
    Поле 3x3;
    Режим для двух игроков (X и O);
    Проверка ввода;
    Определение победителя;
    Определение ничьи;
    Возможность выхода;
    Технологии;
    Python 3;

Запуск:
``` bash
git clone https://github.com/ВАШ_USERNAME/tic-tac-toe.git
cd tic-tac-toe
python Tic-Tac-Toe.py
```

Управление:
    Введите число от 1 до 9: 1 | 2 | 3 4 | 5 | 6 7 | 8 | 9;
    Введите 0 для выхода.

Примечание:
    Проект демонстрирует базовую игровую логику, обработку ввода и структуру кода.
